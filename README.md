###🐾 Paws Shop
An aesthetic inventory management system for pet retail.

Paws Shop is a desktop application designed to streamline how pet supplies are managed and sold. Built with a focus on Object-Oriented Programming (OOP), the system ensures a clean separation of concerns through a modular, role-based architecture.

The Experience:

Manager Portal: Full-scale control to add, update, and oversee the entire inventory.
Cashier Portal: A minimalist interface focused on quick supply searches and sales processing.
The Logic: Built using Java and OOP principles (Inheritance & Encapsulation) to ensure the backend is as organized as the frontend is simple.
